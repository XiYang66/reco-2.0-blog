## Fragment(片段)

- 在vue2中，组件必须有一个跟标签
- 在vue3中，组件可以没有跟标签，内部会将多个标签包含在一个Fragment虚拟元素中

```js
<template>
    <h2>aaaa</h2>
    <h2>aaaa</h2>
</template>
```

## Teleport(瞬移)

- Teleport提供了一种干净的方式，让组件在html页面外特定的标签下插入显示

```js
<teleport to="body">
  <div v-if="modalOpen" class="modal">
    <div>
      I'm a teleported modal! 
      (My parent is "body")
      <button @click="modalOpen = false">
        Close
      </button>
    </div>
  </div>
</teleport>
```

## Suspense(不确定)

- 异步组价在加载期间显示的东西

```js
<Suspense>
  <template #default>
      <AsyncComponent/>
  </template>
  
  <template v-slot:fallback>
    <h2>loading</h2>
  </template>
</Suspense>
```
