## 那些改变

1. 打包体积减少
2. 初次渲染更快
3. 内存更小
4. 使用proxy代替defineProperty数据响应式
5. 重写虚拟DOM的实现和Tree-Shaking

## 新增

1. Composition(组合API)
2. setup (配置属性)
   1. computed和watch
   2. ref和reactive
   3. 新的生命周期函数
